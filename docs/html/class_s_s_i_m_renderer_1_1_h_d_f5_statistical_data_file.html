Detailed Description

The HDF5StatisticalDataFile class represents the structure for statistical data loaded from HDF5 file.

Works with Statismo file format based on HDF5. Inherits from StatisticalData and HDF5Wrapper classes.

Examples:
SimpleStatismoModel.cpp.
