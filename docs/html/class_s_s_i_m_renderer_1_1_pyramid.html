Detailed Description

The Pyramid class represents the structure for pyramid.

Pyramid is defined by 5 points - 4 corners (base) and eye (the top of the pyramid). The corners have to be in same plane, but cannot be in same plane as eye. The eye shold be centered between corners. The non-neighboring corners should be orthogonal to each other.

Examples:
ImageMetrics.cpp, IntensityShapeModel.cpp, and SimpleStatismoModel.cpp.
