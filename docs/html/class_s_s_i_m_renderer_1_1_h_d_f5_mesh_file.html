Detailed Description

The HDF5MeshFile class represents the structure for mesh data loaded from HDF5 file.

Works with Statismo file format based on HDF5. Inherits from Mesh and HDF5Wrapper classes.

Examples:
SimpleStatismoModel.cpp.
